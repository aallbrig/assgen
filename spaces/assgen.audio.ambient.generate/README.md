# assgen · Ambient Soundscape Generator

Generate looping stereo ambient soundscapes for game environments using MusicGen Stereo Large.
Part of the [assgen](https://github.com/aallbrig/assgen) pipeline.

**CLI equivalent:** `assgen gen audio ambient generate "eerie cave drips" --wait`
**Model:** [facebook/musicgen-stereo-large](https://huggingface.co/facebook/musicgen-stereo-large)
