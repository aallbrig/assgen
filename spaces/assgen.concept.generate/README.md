# assgen · Concept Art Generator

Generate game concept art from text descriptions using Stable Diffusion XL.
Part of the [assgen](https://github.com/aallbrig/assgen) pipeline.

**CLI equivalent:** `assgen gen visual concept generate "dark fantasy warrior" --wait`
**Model:** [stabilityai/stable-diffusion-xl-base-1.0](https://huggingface.co/stabilityai/stable-diffusion-xl-base-1.0)
