# assgen · Lore Generator

Generate rich, immersive game world lore entries (history, factions, locations, artifacts) using Phi-3.5-mini.
Part of the [assgen](https://github.com/aallbrig/assgen) pipeline.

**CLI equivalent:** `assgen gen support narrative lore generate "steampunk empire" --wait`
**Model:** [microsoft/Phi-3.5-mini-instruct](https://huggingface.co/microsoft/Phi-3.5-mini-instruct)
