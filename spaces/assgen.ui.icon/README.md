# assgen · UI Icon Generator

Generate game UI icons from text descriptions using SDXL.
Supports Fantasy RPG, Sci-Fi, Minimalist Flat, Pixel Art, and Material Design styles.
Part of the [assgen](https://github.com/aallbrig/assgen) pipeline.

**CLI equivalent:** `assgen gen visual ui icon "health potion" --style "Fantasy RPG" --wait`
**Model:** [stabilityai/stable-diffusion-xl-base-1.0](https://huggingface.co/stabilityai/stable-diffusion-xl-base-1.0)
